# Digital Pet Project

A cute and goofy digital pet that runs on Pi400 during livestreams with nostalgic title sequences and system metric reactivity.

## Overview
This project creates a reactive digital pet that responds to system metrics like CPU temperature and usage, featuring nostalgic title sequences inspired by pop culture references like Final Fantasy VIII intros.

## Features
- Reacts to system metrics (CPU, temperature, memory usage)
- Nostalgic title sequences with pop culture references
- Optimized for Pi400 hardware
- Engaging for livestream audiences
